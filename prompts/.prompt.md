# Transformar Paleta de Cores: Verde → Roxo

## Objetivo

Transformar o esquema de cores da aplicação Dica da Amazônia de **verde** para **roxo**, mantendo toda a funcionalidade intacta.

## Use a Skill: Frontend Design

**Importante:** Use a `frontend-design` skill para garantir alta qualidade visual e design profissional no resultado.

## Detalhes da Mudança

### Cores Atuais (Verde)

- **Cor Primária:** `#4CAF50` (Verde)
- **Cor Secundária:** `#45a049` (Verde mais escuro)
- **Tons relacionados:** Diversos tons de verde

### Cores Alvo (Roxo)

- **Cor Primária:** `#7C3AED` (Roxo vibrante)
- **Cor Secundária:** `#6D28D9` (Roxo mais escuro)
- **Cor de Hover/Ativa:** `#8B5CF6` (Roxo mais claro)
- **Manter:** Brancos, cinzas neutros e acentos do design atual

## Arquivos a Modificar

### Frontend

- `frontend/src/styles/globals.css` - Variáveis de cor global
- `frontend/src/styles/variables.css` - Definições de variáveis CSS
- `frontend/src/App.tsx` - Se houver cores inline
- `frontend/src/components/` - Componentes que usam cores verde

### Paleta de Referência

```
Roxo Principal:     #7C3AED
```

## Requisitos Adicionais

1. Manter a mesma estrutura e funcionalidade
2. Garantir contraste de acessibilidade (WCAG AA)
3. Manter consistência visual entre desktop e mobile
4. Preservar animações e transições existentes
5. Testar no navegador para garantir renderização correta

## Notas

- O verde é usado para indicar sucesso/ações positivas - considere manter essa semântica com o roxo
- Verificar se existem componentes com tema escuro que também precisem atualização
- Atualizar qualquer ícone ou imagem que tenha tons de verde se aplicável
